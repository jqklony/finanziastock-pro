# FinanziaStock Pro

Plataforma Todo-en-Uno de Inventario Inteligente, Facturación, Cobranza y Contabilidad con IA para PYMES y Distribuidores Mayoristas.

## Módulos

1. 📦 **Inventario Inteligente** — Predicción de demanda, punto de reorden automático
2. 🛒 **Compras** — Órdenes sugeridas por IA, comparador de proveedores
3. 💰 **Facturación DIAN** — Facturación electrónica, portal B2B
4. 📋 **Cobranza Inteligente** — Scoring ML, cobro automático WhatsApp + link de pago
5. 📊 **Contabilidad IA** — Conciliación bancaria automática, estados financieros en tiempo real
6. 🤖 **IA Transversal** — 8 modelos ML, predicción de flujo de caja, chatbot contable

## Stack

- **Frontend:** Next.js 15 + React 19 + Shadcn/ui + Tailwind
- **Backend:** NestJS + Prisma + PostgreSQL
- **IA:** FastAPI + XGBoost + Prophet + GPT-4o mini
- **Integraciones:** Belvo (bancos), Twilio (WhatsApp), Wompi (pagos), Factus (DIAN)

## Setup

```bash
npm install
npx prisma db push
npm run dev
```
